# 🐧 Day 3 – Viewing & Editing Files

## 📖 Theory

### What is a File?

A file stores information such as text, images, programs, or data.

### Why do we edit files in Linux?

Most Linux configuration files are plain text files, making terminal-based editing an essential skill.

### What is a Text Editor?

A text editor allows you to create and modify text files. Today, we used **Nano**, a simple terminal-based text editor.

### Why are there different file viewing commands?

Some commands display the entire file, while others display only a portion of it. This helps when working with files of different sizes.

---

## 💻 Commands Learned

| Command | Description |
|---------|-------------|
| `cat` | Display the entire file. |
| `less` | View a file one page at a time. |
| `head` | Display the first 10 lines of a file. |
| `tail` | Display the last 10 lines of a file. |
| `echo` | Print text or write text into a file. |
| `nano` | Open and edit a file using the Nano editor. |
| `>` | Overwrite a file with new content. |
| `>>` | Append new content to the end of a file. |
