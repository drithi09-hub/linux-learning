# 🐧 Day 2 – Files & Directories

## 📖 Theory

### What is a Path?

A path is the address of a file or directory.

### Absolute Path

An absolute path starts from the root directory (`/`) and specifies the complete location.

**Example**

```text
/home/adithi/Documents/Linux
```

### Relative Path

A relative path starts from your current directory.

**Example**

```text
Documents/Linux
```

### Home Directory

The home directory is your personal working directory.

Shortcut:

```text
~
```

### Parent Directory

The parent directory is one level above your current directory.

Shortcut:

```text
..
```

## 💻 Commands Learned

| Command | Description |
|---------|-------------|
| `cd` | Change the current directory. |
| `mkdir` | Create a new directory. |
| `touch` | Create an empty file. |
| `cp` | Copy files or directories. |
| `mv` | Move or rename files/directories. |
| `rm` | Delete files. |
| `rm -r` | Delete a directory and everything inside it. |
| `rmdir` | Delete an empty directory. |
| `tree` | Display the directory structure in a tree format. |
