# 🐧 Day 8 – Networking Basics

## 📖 Theory

### What is a Network?

A network is a group of devices connected together to communicate and share data.

### What is an IP Address?

An IP address is a unique address assigned to a device on a network.

### What is a Hostname?

A hostname is the name of a computer on a network.

### What is Ping?

`ping` checks whether another device or website is reachable over the network.

---

## 💻 Commands Learned

| Command | Description |
|---------|-------------|
| `ip addr` | Display network interfaces and IP addresses. |
| `hostname` | Display the computer's hostname. |
| `ping` | Test network connectivity. |
| `ss -tuln` | Display listening TCP and UDP ports. |
| `curl` | Display webpage content in the terminal. |
| `wget` | Download files or webpages. |

---

## ⭐ Bonus Command

| Command | Description |
|---------|-------------|
| `hostname -I` | Display only the system's IP address(es). |
