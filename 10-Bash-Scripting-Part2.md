# 🐧 Day 10 – Bash Scripting (Part 2)

## 📖 Theory

### What is an if Statement?

An `if` statement allows a Bash script to make decisions based on conditions.

### What is if-else?

The `else` block executes when the `if` condition is false.

### What are Loops?

Loops repeat a block of code automatically.

- `for` → Fixed number of iterations
- `while` → Repeats while a condition is true

### What are Functions?

Functions group commands together so they can be reused multiple times within a script.

---

## 💻 Commands & Concepts Learned

| Concept | Description |
|---------|-------------|
| `if` | Execute code when a condition is true. |
| `else` | Execute alternate code when the condition is false. |
| `-eq`, `-gt`, `-lt`, etc. | Numeric comparison operators. |
| `=` , `!=` | String comparison operators. |
| `-f` | Check whether a file exists. |
| `-d` | Check whether a directory exists. |
| `for` | Repeat commands a fixed number of times. |
| `while` | Repeat commands while a condition is true. |
| `Functions` | Reusable blocks of code. |
| `exit` | Stop script execution. |
| `echo $?` | Display the previous command's exit status. |
| `/dev/null` | Discard command output. |
