# 🐧 Day 5 – Permissions & Users

## 📖 Theory

### What are Permissions?

Permissions determine who can access a file or directory and what actions they can perform.

There are three types of permissions:

- **Read (r)** – View the contents of a file.
- **Write (w)** – Modify the contents of a file.
- **Execute (x)** – Run the file as a program or script.

### Permission Groups

Permissions are assigned to:

- **User (u)** – File owner
- **Group (g)** – Users in the same group
- **Others (o)** – Everyone else

---

## 💻 Commands Learned

| Command | Description |
|---------|-------------|
| `ls -l` | Display file permissions, owner, and group. |
| `chmod` | Change file permissions. |
| `chown` | Change the owner of a file. |
| `sudo` | Run a command with administrator privileges. |
