# 🐧 Day 6 – Package Management

## 📖 Theory

### What is a Package?

A package is a collection of files required to install and run software on Linux.

### What is APT?

APT (Advanced Package Tool) is Ubuntu's package manager. It is used to search, install, update, upgrade, and remove software packages.

### What is a Repository?

A repository is a trusted online collection of software packages maintained by Ubuntu.

---

## 💻 Commands Learned

| Command | Description |
|---------|-------------|
| `apt search` | Search for a package. |
| `apt show` | Display detailed package information. |
| `sudo apt install` | Install a package. |
| `sudo apt remove` | Remove a package. |
| `sudo apt update` | Refresh the package list. |
| `sudo apt upgrade` | Upgrade installed packages. |
| `which` | Check if a package is installed. |
| `sudo dpkg --configure -a` | Complete interrupted package configuration. |
