# 🐧 Day 1 – Terminal Basics

## 📖 Theory

### What is Linux?

Linux is an operating system that manages the computer's hardware and allows programs to run.

### What is a Terminal?

A Terminal is a window where you type commands instead of clicking buttons.

### What is a Shell?

The Shell reads your commands and tells Linux what to do.

### What is a Directory?

A directory is simply a folder.

### What is the Current Directory?

The current directory is the folder you are currently working in.

# 💻 Commands Learned

| Command    | Description                           |
| ---------- | ------------------------------------- |
| `pwd`      | Shows the current directory.          |
| `ls`       | Lists files and folders.              |
| `ls -l`    | Displays a detailed list.             |
| `ls -a`    | Shows hidden files and folders.       |
| `clear`    | Clears the terminal screen.           |
| `history`  | Displays previously used commands.    |
| `whoami`   | Shows the current username.           |
| `date`     | Displays the current date and time.   |
| `cal`      | Shows the current month's calendar.   |
| `uname`    | Displays the operating system name.   |
| `uname -a` | Displays detailed system information. |
| `hostname` | Shows the computer's hostname.        |
| `man`      | Opens the manual page for a command.  |
| `mkdir`    | Creates a new directory.              |
| `cd`       | Changes the current directory.        |
| `rmdir`    | Removes an empty directory.           |
