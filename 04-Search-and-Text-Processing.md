# 🐧 Day 4 – Search & Text Processing

## 📖 Theory

### What is File Searching?

File searching helps locate files and directories without manually browsing folders.

### What is Text Searching?

Text searching helps find specific words or patterns inside a file.

### What is Text Processing?

Text processing involves analyzing text, such as counting lines, words, and characters.

---

## 💻 Commands Learned

| Command | Description |
|---------|-------------|
| `find` | Search for files and directories. |
| `grep` | Search for specific text inside a file. |
| `wc` | Count lines, words, and characters in a file. |
| `which` | Show the location of a command. |
