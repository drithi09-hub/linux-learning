# 🐧 Day 7 – Process Management

## 📖 Theory

### What is a Process?

A process is a program that is currently running on the system.

### What is a PID?

A PID (Process ID) is a unique number assigned to every running process.

### Why is Process Management Important?

Process management helps monitor system activity, identify running programs, and stop unresponsive processes.

---

## 💻 Commands Learned

| Command | Description |
|---------|-------------|
| `ps` | Display processes running in the current terminal. |
| `ps -e` | Display all running processes. |
| `top` | Monitor running processes in real time. |
| `kill` | Stop a process using its PID. |
| `killall` | Stop process(es) using the process name. |
